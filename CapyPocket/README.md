# CapyPocket React UI

Standalone React + Tailwind mobile UI mockup for a cute capybara wallet app.

## Included

- Home screen
- Wallet screen
- Savings screen
- Profile screen
- Reusable Tailwind components
- Inline SVG capybara mascot

## Run

```bash
npm install
npm run dev
```

## Note

If Vite throws an `input/output error` while running from `/mnt/c/...` inside WSL,
move the project to a Linux filesystem path such as `~/projects/CapyPocket`
and run it there. The production build completed successfully from `/tmp`.
